# Image prompts pour le post LinkedIn Lucivy

---

## Option A — Schéma technique stylisé

A clean, modern technical diagram on a dark background. On the left, a search query "programing languag" with visible typos highlighted in red. In the center, a stylized engine with glowing trigram blocks (3-letter fragments) flowing through it like a pipeline. On the right, the matched result "programming language" with the matched portions highlighted in bright cyan. Subtle connecting lines show the fuzzy matching process. Minimalist flat design, no text labels, developer aesthetic. Colors: dark navy background, cyan and orange accents. 16:9 landscape format for LinkedIn.

## Option B — Logo / identité visuelle

A minimalist logo-style illustration for an open source search engine called "Lucivy". A magnifying glass icon where the lens contains stylized trigram characters (three-letter code fragments) arranged in a circular pattern. The handle of the magnifying glass transitions into a Rust gear cog. Clean vector style, flat design, white on a gradient background from deep purple to dark blue. The word "lucivy" in a clean monospace font below. Square format, suitable for social media.

## Option C — Métaphore visuelle (recherche dans le code)

An isometric 3D illustration showing layers of code text floating in space, with a glowing search beam cutting diagonally through multiple layers, highlighting matching fragments across different lines. The highlighted fragments connect with thin luminous threads showing cross-token matching. Some characters glow orange to represent fuzzy/typo tolerance. Dark background with soft depth of field. Modern tech illustration style, similar to Stripe or Vercel blog illustrations. 16:9 landscape.

## Option D — Comparaison avant/après

A split-screen illustration. Left side labeled "Traditional search" (in muted gray tones): a search bar with "programing languag", below it an empty results page with a sad magnifying glass icon. Right side labeled "Lucivy" (in vibrant colors): the same search query, but below it results light up with highlighted matches, connected by glowing trigram nodes. Clean, modern UI mockup style. Subtle code editor aesthetic in the background. 16:9 landscape for LinkedIn.

## Option E — Abstract / artistique

An abstract digital artwork representing text search. Thousands of tiny glowing letters scattered in 3D space like stars. A bright beam of light passes through them, and where it touches, letters rearrange themselves into readable words, snapping into place. Some letters glow orange (representing fuzzy matches — close but not exact). The overall feeling is of order emerging from chaos. Dark space background, warm cyan and orange light. Cinematic, slightly ethereal. 16:9 landscape.

## Option F — N-grams visuels

A dark background with the word "programming" written large in a clean monospace font. Below it, the word is decomposed into overlapping character trigrams: "pro", "rog", "ogr", "gra", "ram", "amm", "mmi", "min", "ing" — each trigram in its own rounded rectangle, slightly overlapping like sliding windows, glowing in different shades of cyan and blue. Above, a search query "programing" (with typo) is shown with its own trigrams, and thin luminous lines connect matching trigrams between the query and the word — showing visually how trigram intersection finds candidates despite the typo. One trigram pair glows orange to represent the fuzzy mismatch. Clean, minimal, technical. 16:9 landscape for LinkedIn.

## Option G — N-gram grid / constellation

A grid of hundreds of small rounded tiles, each containing a 3-character trigram ("pro", "rog", "ogr", "gra", "the", "lan", "ngu", "uag", "age"...), arranged like a periodic table or constellation map on a dark background. Most tiles are dim gray. A search query at the top activates specific trigrams which light up in bright cyan, forming a visible path or cluster through the grid. The activated trigrams connect with subtle glowing lines, converging toward a result document that lights up on the right side. Feels like a neural network or star map activating. Minimalist flat design, developer aesthetic. 16:9 landscape.

## Notes

- Option A est la plus "tech post LinkedIn" — montre concrètement ce que fait Lucivy
- Option B est utile au-delà du post (logo, GitHub, social)
- Option C est la plus "belle" mais moins explicative
- Option D est la plus claire pour quelqu'un qui ne connaît pas le sujet
- Option E est la plus artistique, bonne pour capter l'attention dans le feed
- On peut combiner : par ex. prendre le concept de D (avant/après) avec le style de E (artistique)
